# 📖 บทนำ

การติดตามการเปลี่ยนแปลงของพืชพรรณเป็นส่วนสำคัญของการศึกษาสิ่งแวดล้อม การใช้ที่ดิน และการจัดการทรัพยากรธรรมชาติในระดับภูมิภาคและระดับโลก ปัจจุบันข้อมูลภาพถ่ายดาวเทียมถูกนำมาใช้ในการวิเคราะห์การเปลี่ยนแปลงของพื้นที่สีเขียวอย่างแพร่หลาย เนื่องจากสามารถครอบคลุมพื้นที่ขนาดใหญ่และติดตามการเปลี่ยนแปลงได้ในหลายช่วงเวลา

ดัชนีที่นิยมใช้ในการวิเคราะห์พืชพรรณคือ **Normalized Difference Vegetation Index (NDVI)** ซึ่งคำนวณจากคลื่น **Near Infrared (NIR)** และ **Red**

สูตรคำนวณ NDVI

NDVI = (NIR − RED) / (NIR + RED)

ค่า NDVI มีช่วงตั้งแต่ **-1 ถึง 1**

| ค่า NDVI | ความหมาย |
|----------|-----------|
| < 0 | น้ำ หรือพื้นที่สิ่งปลูกสร้าง |
| 0 – 0.2 | ดินเปล่า |
| 0.2 – 0.5 | พืชพรรณปานกลาง |
| > 0.5 | พืชพรรณหนาแน่น |

ในการศึกษานี้ใช้ข้อมูล **Sentinel-2** เพื่อวิเคราะห์ค่า NDVI ของจังหวัดจันทบุรีในช่วงปี

- 2020  
- 2022  
- 2024

- การเปลี่ยนแปลงของ NDVI (NDVI Change Detection)
- ความสัมพันธ์ระหว่าง NDVI และระดับความสูงของพื้นที่

## 2 การคำนวณค่า NDVI

ใช้ Band ของ Sentinel-2

- **B8 = Near Infrared**
- **B4 = Red**

ตัวอย่างโค้ด

```python
def ndvi(img):
    return img.normalizedDifference(["B8", "B4"]).rename("NDVI")
```

จากนั้นคำนวณ NDVI สำหรับ

- NDVI 2020
- NDVI 2022
- NDVI 2024

---

## 3 การวิเคราะห์การเปลี่ยนแปลง NDVI

คำนวณค่าการเปลี่ยนแปลงของ NDVI

ΔNDVI = NDVI(ปีใหม่) − NDVI(ปีเก่า)

เปรียบเทียบ

- 2020 → 2022  
- 2022 → 2024  

แบ่งระดับการเปลี่ยนแปลง

| ค่า ΔNDVI | การแปลผล |
|-----------|-----------|
| < -0.3 | ลดลงมาก |
| -0.3 ถึง -0.15 | ลดลง |
| -0.15 ถึง -0.05 | ลดลงเล็กน้อย |
| -0.05 ถึง 0.05 | ไม่เปลี่ยนแปลง |
| 0.05 ถึง 0.15 | เพิ่มขึ้นเล็กน้อย |
| > 0.15 | เพิ่มขึ้นมาก |

# 📊 ผลการศึกษา

## ความสัมพันธ์ระหว่าง NDVI และระดับความสูง

*รูปที่ 1 ความสัมพันธ์ระหว่างค่า NDVI และระดับความสูงในจังหวัดจันทบุรี*

จากผลการวิเคราะห์พบว่า **ค่า NDVI มีแนวโน้มเพิ่มขึ้นตามระดับความสูงของพื้นที่**

ตัวอย่างค่าเฉลี่ย NDVI

| ระดับความสูง (เมตร) | Mean NDVI |
|------------------|-----------|
| -100–0 | 0.25 |
| 0–100 | 0.54 |
| 100–200 | 0.62 |
| 200–300 | 0.65 |
| 300–400 | 0.78 |

| 500–600 | 0.82 |

พื้นที่ที่มีความสูง **0–100 เมตร** มีค่า NDVI ประมาณ **0.54**  
ในขณะที่พื้นที่ที่มีความสูงมากกว่า **1000 เมตร** มีค่า NDVI สูงถึงประมาณ **0.85**

แสดงให้เห็นว่า **พื้นที่ภูเขามีพืชพรรณหนาแน่นกว่าพื้นที่ราบ**

## การกระจายตัวของการเปลี่ยนแปลง NDVI

*รูปที่ 2 Histogram ของการเปลี่ยนแปลง NDVI ระหว่างปี 2022 และ 2024*

จาก Histogram ของ ΔNDVI พบว่า

- ค่าการเปลี่ยนแปลงส่วนใหญ่กระจุกอยู่ใกล้ **0**
- แสดงว่าพื้นที่ส่วนใหญ่มี **การเปลี่ยนแปลงของพืชพรรณไม่มาก**

อย่างไรก็ตาม

- มีบางพื้นที่ที่มีค่า **ΔNDVI < -0.3** ซึ่งบ่งชี้ถึงการลดลงของพืชพรรณ
- และบางพื้นที่มีค่า **ΔNDVI > 0.15** แสดงถึงการเพิ่มขึ้นของพืชพรรณ

โดยรวมแล้ว การเปลี่ยนแปลง NDVI ในช่วง **2022–2024** อยู่ในระดับ **เล็กน้อยถึงปานกลาง**
