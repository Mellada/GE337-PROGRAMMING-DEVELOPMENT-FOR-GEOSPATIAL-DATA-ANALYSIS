# 📖 Introduction

การติดตามการเปลี่ยนแปลงของพืชพรรณเป็นหนึ่งในวิธีสำคัญในการศึกษาสภาพแวดล้อม การใช้ที่ดิน และการจัดการทรัพยากรธรรมชาติในระดับภูมิภาคและระดับโลก ภาพถ่ายดาวเทียมช่วยให้สามารถตรวจสอบการเปลี่ยนแปลงของพื้นที่สีเขียวในช่วงเวลาต่าง ๆ ได้อย่างมีประสิทธิภาพ

หนึ่งในดัชนีที่นิยมใช้ในการวิเคราะห์พืชพรรณคือ **Normalized Difference Vegetation Index (NDVI)** ซึ่งคำนวณจากคลื่น **Near Infrared (NIR)** และ **Red** จากข้อมูลดาวเทียม

สูตรคำนวณ NDVI

NDVI = (NIR − RED) / (NIR + RED)

ค่า NDVI อยู่ในช่วง **-1 ถึง 1**

| NDVI Value | Meaning |
|------------|--------|
| < 0 | Water / Built-up area |
| 0 – 0.2 | Bare soil |
| 0.2 – 0.5 | Moderate vegetation |
| > 0.5 | Dense vegetation |

ในการทดลองนี้ใช้ข้อมูล **Sentinel-2** เพื่อวิเคราะห์ NDVI ของจังหวัดจันทบุรีในช่วงเวลา

- 2020  
- 2022  
- 2024  

รวมทั้งวิเคราะห์

- การเปลี่ยนแปลงของ NDVI (NDVI Change Detection)
- ความสัมพันธ์ระหว่าง NDVI กับระดับความสูงของพื้นที่

## 2 NDVI Calculation

ใช้ Band

- **B8 = Near Infrared**
- **B4 = Red**

ตัวอย่างโค้ดคำนวณ NDVI

```python
def ndvi(img):
    return img.normalizedDifference(["B8", "B4"]).rename("NDVI")
```

จากนั้นคำนวณ NDVI สำหรับแต่ละปี

- NDVI 2020
- NDVI 2022
- NDVI 2024

และแสดงผลเป็นแผนที่

---

## 3 NDVI Change Detection

คำนวณการเปลี่ยนแปลงของ NDVI

ΔNDVI = NDVI(new) − NDVI(old)

เปรียบเทียบช่วงเวลา

- 2020 → 2022  
- 2022 → 2024  

แบ่งระดับการเปลี่ยนแปลงดังนี้

| ΔNDVI | Interpretation |
|------|---------------|
| < -0.3 | Decrease strong |
| -0.3 – -0.15 | Decrease |
| -0.15 – -0.05 | Slight decrease |
| -0.05 – 0.05 | No change |
| 0.05 – 0.15 | Slight increase |
| > 0.15 | Increase strong |

3. วิเคราะห์แนวโน้มของค่า NDVI ตามระดับภูมิประเทศ

# 🧠 Lab Questions

## 1 ทำไมการวิเคราะห์ NDVI จากภาพถ่ายดาวเทียมหลายช่วงเวลาจึงมีความสำคัญ?

การวิเคราะห์ NDVI หลายช่วงเวลาช่วยให้สามารถติดตามการเปลี่ยนแปลงของพืชพรรณในพื้นที่ได้ เช่น การเพิ่มขึ้นหรือลดลงของพื้นที่สีเขียว การเปลี่ยนแปลงของพื้นที่เกษตร หรือการสูญเสียพื้นที่ป่า นอกจากนี้ยังช่วยให้เห็นแนวโน้มของการเปลี่ยนแปลงในระยะยาวและลดผลกระทบจากปัจจัยชั่วคราว เช่น สภาพอากาศหรือฤดูกาล

## 2 พื้นที่ที่มีการเปลี่ยนแปลง NDVI อย่างมีนัยสำคัญควรได้รับการตรวจสอบเพิ่มเติมอย่างไร?

พื้นที่ที่พบว่ามีการเปลี่ยนแปลง NDVI อย่างมากควรตรวจสอบเพิ่มเติมโดย

- เปรียบเทียบภาพดาวเทียมหลายช่วงเวลา
- ตรวจสอบข้อมูลการใช้ที่ดิน
- ใช้ภาพความละเอียดสูง เช่น Google Earth
- ทำการสำรวจภาคสนาม

เพื่อยืนยันว่าการเปลี่ยนแปลงนั้นเกิดจากการเปลี่ยนแปลงของพืชพรรณจริง

## 3 ปัจจัยใดที่อาจทำให้ค่า NDVI เปลี่ยนแปลงโดยไม่เกี่ยวข้องกับพืชพรรณ?

- เมฆและเงาของเมฆ

- ฤดูกาล
- มุมแสงอาทิตย์
- ความคลาดเคลื่อนของเซนเซอร์ดาวเทียม
- ฝุ่น ควัน หรือหมอก

ปัจจัยเหล่านี้อาจทำให้ค่า NDVI เปลี่ยนแปลงแม้ว่าพืชพรรณจะไม่ได้เปลี่ยนแปลงจริง

## 4 หากต้องการเพิ่มความแม่นยำของการวิเคราะห์ NDVI ควรใช้เทคนิคอะไรเพิ่มเติม?

DEM ช่วยให้สามารถวิเคราะห์ความสัมพันธ์ระหว่างพืชพรรณกับลักษณะภูมิประเทศ เช่น ความสูงและความลาดชัน ในการศึกษานี้พบว่าค่า NDVI เพิ่มขึ้นตามระดับความสูง ซึ่งแสดงว่าพื้นที่ภูเขามีพืชพรรณหนาแน่นกว่าพื้นที่ราบ ดังนั้น DEM ช่วยให้เข้าใจปัจจัยทางภูมิประเทศที่มีผลต่อการกระจายตัวของพืชพรรณ
